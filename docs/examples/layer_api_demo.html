layer_api demo

Worked example for atlas-core/core/layer_api.js — exercises every endpoint pages need to consume action-pipeline layers. Open this page via the atlas server (so same-origin fetch() works), or set the base URL below and run it from file://.

1. Base URL (optional)

When the page is served by atlas_server.py, leave empty.

2. List layers — GET /api/layers

layer_idlayer_typeschema_version stagedataset_idstatuscreated_at

3. Fetch one envelope — GET /api/layers/{layer_id}




4. Submit an action — POST /api/actions



  Paste a manifest matching action_manifest.schema.json. The
  action_id is auto-filled to a fresh schema-conformant id; edit if
  you want to retry a known one. atlas resolves the dispatcher
  (precedence: ?atlas=… > manifest.atlas_id >
  master_config.atlas.active_atlas).




  
  
  









5. Read an action's log entry — GET /api/actions/{action_id}
