Graph builder — five-column layer graph editor

Five columns, one per node type from vocabulary/node_types.tsv: sets / filters / layers / analyses / hooks. Each column lists the items registered in the workspace. Click a node to add it to the current graph (a small IN GRAPH badge appears). Then click two in-graph nodes — the second click draws an edge with the current edge type. Edges are validated live against vocabulary/edge_rules.tsv: green = valid, red = invalid. Click an edge to delete.
